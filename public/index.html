关于这个 Blog

idea 来源于 xyix的自制blog

部署在 Github Page 上，Gitee 和 Netlify 上也有镜像

这个博客的构建工具经过多次迭代，目前的所有东西都是基于 Scheme 和 S-表达式的，甚至没有用 markdown。 目前的 Scheme 环境是 Racket，较为方便，以后如果出现性能问题再改成 Chez-scheme。

现在的一篇文章长这样：

($article
  ()

  (h1 关于这个 Blog)
  (p idea 来源于 ,($link https://xyix.github.io xyix的自制blog))
  (p 这个博客的构建工具经过多次迭代，目前的所有东西都是基于 Scheme 和 S-表达式的，甚至没有用 markdown。
     目前的 Scheme 环境是 Racket，较为方便，以后如果出现性能问题再改成 Chez-scheme。))
